Control Flow, Loops and Arrays

This week we cover the building blocks of algorithmic thinking in Java: loops, arrays, and method definitions.

For loops

for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

While loops

int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

Arrays

int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // 1
