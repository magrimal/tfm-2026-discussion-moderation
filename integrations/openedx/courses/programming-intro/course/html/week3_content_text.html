Object-Oriented Programming: Classes and Inheritance

This week introduces object-oriented programming. You will learn to define classes, create objects, and use inheritance to model relationships between types.

Defining a class

public class Animal {
    private String name;

    public Animal(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }
}

Inheritance

public class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }

    public void bark() {
        System.out.println(getName() + " says: Woof!");
    }
}
