Boolean Operators in Java

In this section we cover the three fundamental boolean operators: AND, OR, and NOT. These operators let you combine conditions and control program flow.

AND operator (&&)

Returns true only when both operands are true.

boolean a = true;
boolean b = false;
System.out.println(a && b); // false

OR operator (||)

Returns true when at least one operand is true.

System.out.println(a || b); // true

NOT operator (!)

Inverts a boolean value.

System.out.println(!a); // false
