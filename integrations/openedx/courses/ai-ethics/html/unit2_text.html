Fairness Criteria and Trade-offs

Fairness in machine learning is not a single property — it is a family of mathematically incompatible criteria. Understanding the differences is essential before evaluating any system.

Demographic parity

A classifier satisfies demographic parity if positive prediction rates are equal across groups. This does not account for base rate differences between groups.

Equalized odds

Requires equal true positive rates and equal false positive rates across groups. This can conflict with demographic parity when base rates differ.

Individual fairness

Similar individuals should receive similar predictions. Requires a definition of similarity, which can itself encode bias.

A landmark result (Chouldechova, 2017; Kleinberg et al., 2016) shows that demographic parity, equalized odds, and calibration cannot all hold simultaneously when base rates differ between groups. Any fairness definition involves trade-offs.
