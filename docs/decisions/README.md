# Registro de decisiones

Este directorio contiene el registro de decisiones relevantes tomadas durante el desarrollo del TFM.

El objetivo es documentar de forma ligera las decisiones técnicas y de diseño que afectan al proyecto, junto con su contexto y justificación.

No todas las decisiones se documentan como ADRs formales; solo aquellas que tienen impacto significativo en la arquitectura, el enfoque técnico o el alcance del proyecto.
