# Literatura

Revisión de literatura del TFM.

## Contenido

- [papers.csv](papers.csv) — tabla resumen de todos los papers
  revisados (campos: autor, año, tipo, contexto, método, hallazgo
  clave, limitaciones).
- [evidence-by-hypothesis.md](evidence-by-hypothesis.md) — evidencia
  concreta (citas, estadísticas, páginas) organizada por las hipótesis
  del TFM.
- `papers/` — copias locales de los PDFs revisados (gitignored).
  Naming: `author-year-short-title.pdf`.
